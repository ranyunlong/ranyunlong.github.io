使用@lemonc/cli 快速创建electron项目

目录

  1. 简介
  2. 安装
  3. 使用

如果你可以建一个网站，你就可以建一个桌面应用程序。 Electron 是一个使用 JavaScript, HTML 和 CSS 等 Web 技术创建原生程序的框架，它负责比较难搞的部分，你只需把精力放在你的应用的核心上即可。

简介

@lemonc/cli 是一个快速构建electron项目的脚手架，它采用的是webpack构建，electron也比较适合单页面应用spa的应用开发，所以，@lemonc/cli 使用了webpack 当然也不会少了vue 和 react。项目集成了vue和react。使用electron-builder来打包应用。

使用

创建vue脚手架应用

> lemonc init vue myApp

创建react脚手架应用

> lemonc init react myApp
